# email_spam_classification
This project is a classroom assignment about spam email classification using Machine Learning algorithms.
The objective of this project is to classify emails into two categories:

- Spam
- Not Spam (Ham)

The models were implemented manually for learning purposes and trained on a text dataset.

## Models Used
1. Logistic Regression (Chi Nguyen Phuong)

Logistic Regression was implemented from scratch and used for binary classification of emails.

Result:Accuracy: 80%
2. Multinomial Naive Bayes (MNB) (Teammate)

Multinomial Naive Bayes was implemented and evaluated on the same dataset.

Result: Accuracy: 93.8%
